Representative Proposal Form:

 

Add Representative Proposal

 

Based on the user login level - office details are displayed in the form i.e., Head Office, Circle Office, Division Office or Subdivision Office.

 

         Head office Login: Head office details are automatically displayed in read only mode. The user has to select the circle, division, sub division for which the representation is being made.

         Circle Office Login: The head office and circle office details are automatically displayed in read only mode. The user has to select division, sub division for which the representation is being made.

         Division Office Login: The head office, circle office and division office details are automatically displayed in read only mode. The user has to select sub division for which the representation is being made.

         Sub Division Office Login: The head office, circle office and division office details are automatically displayed in read only mode.

If the Representative Code selected is MLA Or MP then

       Enter the CMP.No.

If User has selected Endorsed by as Others then

       Enter the details in the field specified.

 

If User has selected Endorsed by as Minister then

       Select Minister For

If User has selected Minister for as Others then

Enter the details in the field specified.

Manage Representative Proposal:

 

Based on the user login level - office details are displayed in the form i.e., Head Office, Circle Office, Division Office or Subdivision Office.

 

         Head office Login: Head office details are automatically displayed in read only mode. The user has to select the circle, division, sub division for which the representation is being made.

         Circle Office Login: The head office and circle office details are automatically displayed in read only mode. The user has to select division, sub division for which the representation is being made.

         Division Office Login: The head office, circle office and division office details are automatically displayed in read only mode. The user has to select sub division for which the representation is being made.
