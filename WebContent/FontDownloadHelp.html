How to Download and Install India Rupee Font

Fordian Technologies has created the new Indian rupee font for free download.
Its simple to use. Please do the following Steps:

1. Download the Rupee font (.ttf file) by Clicking HERE
2. To install the font in your computer Copy the downloaded font into fonts folder
    (typically C:\Windows\Fonts for Windows XP (or) C:\WinNT\Fonts for Others).
3. The grave acent symbol (`) (the key just above “tab” button in your keyboard) is used to type the new Rupee symbol.
4. Select “Rupee” font from the drop down list of your fonts in your document .
